    Como executar no Apps Script
    Passo a passo 1. Clique em "🔧 Corrigir e Copiar" — o script é corrigido e copiado para o clipboard
    2. Abra script.google.com → crie um novo projeto
    3. Cole o script no Code.gs → salve (Ctrl+S)
    4. Execute a função principal (ex: gerarGuia()) → autorize as permissões
    5. O documento aparece na pasta "Plataforma de Estudos MEA" no Drive

    Simulado CEBRASPE — 25 Questões

    Calibrada para CONSULPLAN: C=34%, B=26%, A=24%, D=16%, distratores por absolutização.
    Lista Ensino Médio — 20 Questões
    Lista didática para ensino médio. Linguagem clara, progressão de dificuldade 1 a 5.

    Prompt para análise total de qualquer banca: perfil cognitivo, anatomia, distratores, estratégias e validação cruzada.
